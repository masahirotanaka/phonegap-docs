contactFindOptions
==================

 `contact.find` での連絡先検索にオプションで指定するパラメータ。検索のフィルターに使用します。

    { 
		filter: "",
		multiple: true,
	};

オプション
--------------

- __filter:__ 検索フィルターに使用する文字列 _(DOMString)_ (Default: "")
- __multiple:__ `contact.find` メソッドが複数の連絡先を返すかどうかを指定する _(Boolean)_ (Default: false)
