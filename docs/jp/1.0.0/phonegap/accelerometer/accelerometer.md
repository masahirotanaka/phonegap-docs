加速度センサー
================

> x, y, z-軸におけるデバイスの傾きをキャプチャします。

メソッド
---------

- accelerometer.getCurrentAcceleration
- accelerometer.watchAcceleration
- accelerometer.clearWatch

引数
---------

- accelerometerSuccess
- accelerometerError
- accelerometerOptions

オブジェクト (読込専用)
-------------------

- Acceleration
