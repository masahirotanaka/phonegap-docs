Connection
==========

> `connection` デバイスの通信手段(wifi, 3G, etc...)に関する情報を提供するオブジェクトです。

このオブジェクトは `navigator.network` からアクセスすることができます。

プロパティ
----------

- connection.type

定数
---------

- Connection.UNKNOWN
- Connection.ETHERNET
- Connection.WIFI
- Connection.CELL_2G
- Connection.CELL_3G
- Connection.CELL_4G
- Connection.NONE
