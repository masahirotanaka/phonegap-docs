CaptureAudioOptions
===================

> オーディオキャプチャのオプション設定の情報をカプセル化したオブジェクトです。

プロパティ
----------

- __limit:__ 一度のキャプチャセッションで録音できるオーディオクリップの最大数 (デフォルト: 1)
- __duration:__ オーディオクリップの録音最大秒数
- __mode:__ 選択されたオーディオモード。 `capture.supportedAudioModes` で定義されたいずれかの値にマッチしなければならない。

使用例
-------------

    // 一度のセッションでの録音の秒数を10秒以下、録音できるクリップ数を3つに制限。
    var options = { limit: 3, duration: 10 };

    navigator.device.capture.captureAudio(captureSuccess, captureError, options);

Androidに関する注意点
--------------

- __duration__ はサポートされていません。レコード秒数はプログラムからは制御できません。
- __mode__ はサポートされていません。オーディオの録音フォーマットはプログラムからは変更できません。録音はAMR(audio/amr)フォーマットでエンコードされます。

BlackBerry WebWorksに関する注意点
--------------------------

- __duration__ はサポートされていません。レコード秒数はプログラムからは制御できません。
- __mode__ はサポートされていません。オーディオの録音フォーマットはプログラムからは変更できません。録音はAMR(audio/amr)フォーマットでエンコードされます。

iOSに関する注意点
--------------------

- __limit__ はサポートされていません。一度の録音で一回のキャプチャのみ可能です。
- __mode__ はサポートされていません。オーディオの録音フォーマットはプログラムからは変更できません。録音はWAV(audio/wav)フォーマットでエンコードされます。
